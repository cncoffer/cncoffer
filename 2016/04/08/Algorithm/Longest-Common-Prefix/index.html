Longest Common Prefix

Question

Write a function to find the longest common prefix string amongst an array of strings.

翻译：写一个函数，用于找出一组字符串的最长公共前缀。

Answer

暴力算法

首先想到的方法是从前两个字符串得到一个公共前缀，然后用得到的公共前缀一个一个向后比较更新，最后得到一个最长公共前缀。
时间复杂度应该为o(n*l),l is avg length of string.
代码如下：

class Solution
{
public:
    Solution();
    ~Solution();
public:
    string longestCommonPrefix(vector<string>& strs) {
        string longest_prefix;
        int size = strs.size();

        longest_prefix.clear();
        if (size == 0) { // 排除不存在字符串和只存在一个字符串的特殊情况
            return longest_prefix;
        }
        else if (size == 1) {
            return strs[0];
        }

        longest_prefix = CommonPrefix(strs[0], strs[1]); // 先取前两个字符串的公共前缀作为比较对象
        int i;
        i = 2;
        while (i < size) {
            longest_prefix = CommonPrefix(strs[i], longest_prefix); // 用公共前缀与每一个字符比较,并在比较的同时不停更新
            i++;
        }

        return longest_prefix;
    }
private:
    string CommonPrefix(string str1, string str2) { // 返回两个指定字符串的公共前缀
        string prefix;
        int str1_length, str2_length;

        str1_length = str1.length();
        str2_length = str2.length();

        prefix.clear();
        if (str1_length == 0 || str2_length == 0) { // 排除空字符串的情况
            return prefix;
        }

        int index, length;
        index = 0;
        length = (str1_length < str2_length) ? str1_length : str2_length; // 计算短字符串长度,这也是公共前缀的最大可能长度
        while (index < length) {
            if (str1[index] != str2[index]) { // 若有字符不同,则返回
                break;
            }
            index++;
        }

        return str1.substr(0, index); // 此时index指向的是不同的或超出长度的那个字符,因此这里返回的字串的字符个数(即长度)应当是index
    }
};

暴力算法改进

然后想到比较的次数应当可以减少。

比如对于有四个字符串的输入，a[0],a[1]比较，a[2],a[3]比较，然后得到的结果再比较，仅比较了3次，比上面的方法少了一次。

上面说的这个改进几乎不能减少时间复杂度，还增加了编码的复杂度，是我想太多~~
