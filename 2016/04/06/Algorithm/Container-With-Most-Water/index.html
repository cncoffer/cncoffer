Container With Most Water

Question

Given n non-negative integers a1, a2, …, an, where each represents a point at coordinate (i, ai). n vertical lines are drawn such that the two endpoints of line i is at (i, ai) and (i, 0). Find two lines, which together with x-axis forms a container, such that the container contains the most water.
Note: You may not slant the container.

翻译：给出n个非负整数a1,a2,……,an,这些数代表了一个点的坐标(i,ai). 在坐标轴上根据(i,ai)和(i,0)为端点画了n条垂直线段。要求寻找两条线，他们和x轴组成的容器能容纳最多的水。
提示：你不应倾斜容器。

Answer

暴力算法

惯例，首先想到的肯定是双重循环的暴力算法啦，将所有容器能容纳的水都计算出来，找出最大的就是了。
代码如下：

int maxArea(vector<int>& height) {
    int size = height.size();
    int i, j;

    if (size <= 1) {
        return 0;
    }

    unsigned int max_area, area;
    int min_height;
    max_area = 0;
    for (i = 0; i < size; i++) {
        for (j = i + 1; j < size; j++) {
            min_height = (height[i]<height[j]) ? height[i] : height[j];
            area = (j - i)*min_height;
            if (area > max_area) {
                max_area = area;
            }
        }
    }
    return max_area;
}

不过并没有通过测试，因为超时。
这一算法的时间复杂度是O(n^2),明显是一个比较糟糕的算法，下面对其进行改进。


从两端收缩

主要是参考了csdn上的资料，我是真想不到这么好的方法。
原理如下：

  1. 假设i,j之间的容器最大，即(i,a[i])和(j,a[j])组成的容器最大，那么有如下两条性质
  2. 对于k,若k>j,则a[k]a[j],则a和k之间的容器必定比i和j之间的大。
  3. 反之,若k<i,则a[k]<a[i].
  4. 也就是说，从两边收缩，确定了端点x,y之后，比他们大的容器必定在[x,y]中，且a[max_left]>=a[x]&&a[max_right]>=a[y].

因此可以确定从两端收缩，不停更新端点值的算法。再具体的说明可以查看参考资料。

代码如下：

class Solution2
{
public:
    Solution2();
    ~Solution2();
public:
    int maxArea(vector<int> &height) {
        int res = 0;
        int size = height.size();
        int left = 0, right = size - 1;
        while (left<right)
        {
            res = max(res, min(height[left], height[right])*(right - left));
            if (height[left]<height[right]) // 从较低的边开始收缩
            {
                int k = left;
                while (k<right&&height[k] <= height[left]) // 仅比较比两端高的边
                    k++;
                left = k;
            }
            else
            {
                int k = right;
                while (k>left&&height[k] <= height[right]) // 仅比较比两端高的边
                    k--;
                right = k;
            }
        }
        return res;
    }
private:
    int min(int a, int b) {
        return a < b ? a : b;
    }
    int max(int a, int b) {
        return a > b ? a : b;
    }
};
