Palindrome Number

Question

Determine whether an integer is a palindrome. Do this without extra space.

Answer

首先想到的是将最高位和最低位的数字取下来比较，然后去次高位和次低位的比较，以此类推。知道比较到中间数字，得出结果。
有几点需要注意：

  1. 负数如何处理？ 根据测试的要求，负数不算。
  2. 是否会有溢出？ 处理的时候控制不要出现比x大的数就不会溢出。

代码如下：

bool isPalindrome(int x) {
    unsigned int temp;
    int length;
    if (x < 0) {
        return false;
    }
    else if (x == 0) {
        return true;
    }

    length = 0;
    temp = x;
    while (temp > 0) {
        temp /= 10;
        length++;
    }

    unsigned int front_index, behind_index;
    int front, behind;
    front_index = (int)pow(10, length - 1 ); // front_index 有可能溢出
    behind_index = 100;
    front = x / front_index;
    behind = x % 10;
    if (behind != front) {
        return false;
    }

    while (behind_index < front_index) {
        behind = (x % behind_index) / (behind_index / 10);
        front = (x % front_index) / (front_index / 10);
        if (behind != front) {
            return false;
        }
        else {
            // 如果将前后数字取走看似方便，但是1000021这样的数判断不出，取了第一个1就变成了21了，因此第一位的数字不能取走
            front_index /= 10;
            behind_index *= 10;
        }
    }
    return true;
}

这里用的就是上面描述的方法，从高位和低位不停的取数字比较，直到比较到中间。
然后说到结果，看起来不错，就不继续研究下去了。
