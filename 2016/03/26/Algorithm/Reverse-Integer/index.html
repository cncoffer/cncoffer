Reverse Integer

Question

Reverse digits of an integer.
反转整型数字。

Example1: x = 123, return 321
Example2: x = -123, return -321

Answer

方法一

首先想到的是将x从个位开始，一个个取下来，然后一个个反序放入另一个int型变量。

y *= 10;
y += x%10;
x /= 10;

如此不停循环，直到取完x的每一位数字。
需要注意检查反转后是否溢出。
代码如下：

#define NUMBER_TEN 10
#define OVER_FLOW_LIMIT  214748364 // (INT_MAX / 10)
int reverse(int x) {
    int flag, result;

    if (x <= 9 && x >= -9) { // 若只有一位，则直接返回
        return x;
    }

    if (x < 0) { // 提取符号位
        flag = -1;
        x = -x;
    }
    else {
        flag = 1;
    }

    result = 0;
    while (x > 0) {
        if (result > OVER_FLOW_LIMIT) { // 检测是否会溢出
            return 0;
        }
        result *= NUMBER_TEN;
        result += （x % NUMBER_TEN）;
        x /= NUMBER_TEN;
    }

    return result * flag;
}

看这个结果似乎还可以，不过感觉还是有可以改进的地方。算法复杂度是O(l)，l表示整数长度。


题外话 !方法二

接着我想到，是否能只在源数字变量x上进行变换呢？而这样是否有提高效率的办法呢？

本来想考虑一下其他方法的，仔细思考了一下。好吧，我承认我傻了，针对这题，int型最大也就2^32，大约10个10进制位，O(l)的时间复杂度，这哪里还有深究算法的必要。

不过长字符串倒是可以用这样的办法，应该能省下比较多的空间，但是字符串取字符又没有取十进制数字这么麻烦了，遇到再说吧~

还是不纠结了-。-
